# Mike Williams Blog

Welcome to the Mike Williams Blog! This blog is dedicated to sharing technical articles and tutorials on various topics, including Piaware and .NET development. 

## Project Overview

This project is built using the Beautiful Jekyll theme and is hosted on GitHub Pages. It serves as a platform for sharing knowledge and insights on technology, programming, and software development.

## Features

- **Technical Articles**: A collection of articles covering topics from Piaware setup to .NET tutorials.
- **Responsive Design**: The blog is designed to be mobile-friendly and accessible on various devices.
- **Easy Navigation**: The navigation structure allows users to easily find articles and resources.

## Getting Started

To set up this project locally, follow these steps:

1. **Clone the Repository**:
   ```
   git clone https://github.com/yourusername/mike-williams-blog.git
   cd mike-williams-blog
   ```

2. **Install Dependencies**:
   Make sure you have Ruby and Bundler installed. Then run:
   ```
   bundle install
   ```

3. **Run the Jekyll Server**:
   Start the Jekyll server to view the blog locally:
   ```
   bundle exec jekyll serve
   ```

4. **Open in Browser**:
   Navigate to `http://localhost:4000` in your web browser to view the blog.
